- **Agentic and AI systems** — workflows that combine models, tools, memory, evidence and operational control.

## Current focus

### [VIRA](https://vira.snelabs.space/?lang=en)

A synchronized multiplayer football experience powered by authoritative match data. VIRA turns live match moments into shared challenges with server-owned deadlines, deterministic resolution, append-only event history and optional Solana commitments.

`React` `TypeScript` `Node.js` `SSE` `Event Sourcing` `Playwright` `Solana` `Docker`

### Self-custody wallet R&D

A modular desktop wallet architecture spanning Electron, React, Go and gRPC, with local encrypted vaults and execution paths across Bitcoin, Liquid and Lightning. The project is currently in technical pre-beta and remains private while its release gates are hardened.

`Go` `Electron` `gRPC` `Bitcoin` `Liquid` `Lightning` `SQLite` `Docker`
